## Readme

An empty Rails application. There's nothing to see here.

## Getting Started

1. Install gems:

        bundle install

2. Create the database:

        rake db:create

3. Start the web server:

        rails server

4. Using a browser, go to `http://localhost:3000` and you'll see:
"Welcome aboard: You're riding Ruby on Rails!"
